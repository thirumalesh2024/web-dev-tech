Live Server Setup Documentation

Introduction

This guide explains how to use Visual Studio Code (VS Code) with the Live Server extension to view your HTML files in a browser. It includes information on accessing the local URL http://127.0.0.1:5500/index.html.

Steps to Set Up and Run

  • Install Visual Studio Code from https://code.visualstudio.com/.
  • Open VS Code and install the Live Server extension:
    • Go to the Extensions view (Ctrl+Shift+X or Cmd+Shift+X on Mac).
    • Search for Live Server by Ritwick Dey and click Install.
  • Create a new folder for your project and open it in VS Code.
  • Inside the folder, create a file named index.html and paste your HTML code.
  • Right-click the index.html file and select Open with Live Server.
  • Your browser will open a local URL like http://127.0.0.1:5500/index.html.

Accessing the Local URL

Once Live Server is running, your file will be accessible at http://127.0.0.1:5500/index.html. This URL has the following components:

  • 127.0.0.1: Refers to your local computer (also known as localhost).
  • 5500: The port number used by Live Server (may vary depending on your setup).
  • index.html: The file you are viewing in the browser.

Troubleshooting

  • If the browser doesn't open, ensure Live Server is installed and running.
  • If you see "404 Not Found," ensure the index.html file is saved in the correct directory.
  • If you want to share the page on the same network, replace 127.0.0.1 with your local IP (e.g., 192.168.x.x).
